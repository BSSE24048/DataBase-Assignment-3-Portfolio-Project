A modern, minimal, and futuristic **portfolio website** built using **React + Vite**, showcasing the profile, skills, and projects of a Software Engineering student.

---

## 🎨 Design Philosophy

This portfolio follows a **light-themed futuristic UI** with:

- Clean whitespace & minimal layout  
- Soft gradients and glassmorphism effects  
- Subtle background animations (floating blobs)  
- Smooth transitions and micro-interactions  
- Calm, elegant, and modern feel  

---

## 🚀 Features

- 🌐 **Multi-page navigation** using React Router  
- 🎯 **Hero section** with CV download  
- 📄 **About page** with "Read More / Read Less" toggle  
- 🧠 **Skills section** using animated cards  
- 📦 **Projects section** with reusable components  
- 📬 **Contact form** with success animation  
- 🎨 **Custom CSS theme** (no libraries)  
- 🌌 **Animated background (floating gradients)**  
- ⚡ **Fast performance with Vite**  
- 📱 **Fully responsive design**  

---

## 🧠 React Concepts Used

| Concept | Implementation |
|--------|---------------|
| Components | Navbar, Footer, SkillCard, ProjectCard, ContactForm |
| Props | Passing data to reusable cards |
| useState | About toggle, contact form state |
| useEffect (mount) | Set page title |
| useEffect (dependency) | Track state changes |
| useEffect (no dependency) | Runs on render |
| Routing | React Router DOM |
| Conditional Rendering | Success message, read more |
| List Rendering | Skills & Projects mapping |
